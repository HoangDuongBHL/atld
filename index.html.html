    Kéo thả file Excel vào đây

    Chỉ cần nạp 1 file duy nhất — dashboard sẽ tự động đọc cả 2 sheet bên trong. Dữ liệu được xử lý hoàn toàn trong trình duyệt của bạn, không tải lên máy chủ nào.
